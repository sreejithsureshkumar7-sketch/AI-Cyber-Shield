# AI Cyber Shield

AI Cyber Shield is a full-stack cyber security project using frontend, backend, and AI-style rule-based detection.

## Features
- AI Phishing Link Detector
- AI Password Strength Analyzer
- Suspicious Login Detection
- QR Scam Checker
- Security Dashboard
- AI Security Assistant Chatbot
- Download Security Report
- Dark Cyber Theme

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- AI Logic: JavaScript-based scoring models

## Run Frontend
Open `frontend/index.html` in browser.

## Run Backend
```bash
cd backend
npm install
npm start
```
Backend runs on `http://localhost:5000`.

## Demo URLs
- Backend: http://localhost:5000
- URL Scan API: http://localhost:5000/api/scan-url
